# eos-core

[Saves](https://github.com/gavin-chan/saves)项目的核心，用RESTful实现的接口，有关接口，有更详细的文档说明。

# 设计
1. 流程如下：
    ```
      接收指令----->指令验证----->识别并分发任务--(上传)-->生成存储信息----->cache到数据区加密----->存储数据----->返回成功
                                      |
                                      +-------(下载)-->查询数据----->数据区到cache解密----->记录下载----->返回cache路径
                                      |
                                      +-------(查询)-->查询数据----->返回数据
    ```
    其中，任意一个步骤发生错误将返回。
